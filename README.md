## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

src/api 项目中程序的接口以及调用接口的方法文件夹
        /interface.js 接口存放文件
        /aConfig.js a模块的接口获取方法
        /bConfig.js b模块的接口获取方法

src/comassets 项目中公共的静态文件的文件夹
               /image 公共的图片资源
               /font 公共的字体样式资源
               /style 公共的css样式资源 

src/commethod 项目中公共的方法的文件夹

src/components 项目中公共组件的文件夹

src/nativemethod 项目中调用原生方法的文件夹

src/router 项目中存放路由的文件夹

src/store 项目中存放状态的文件夹

src/utils 项目中辅助工具方法的文件夹

src/view 项目中单独组件的页面的文件夹
